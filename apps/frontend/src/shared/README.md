Слой в свободной форме. Структура файлов в слое не имеет строгих требований, она может быть любой.
Из этого слоя могут импортировать все остальные слои в любых местах. 
Этот слой как кладовая, тут весь инвентарь разработчика.

# Файловая структура среза
В приложении принята такая структура

- `lib/` - Библиотеки и методы, общие на весь проект
  В срезе lib одно строгое правило, все что касается какого либо инструмента
  должно находиться в одном срезе. TS типы и интерфейсы должны находиться там же, 
  где они реализуются или используются. На примере композиции `useApi()`, типы вроде `ApiErrorResponse`
  должны находиться в том же срезе что, и реализация `useApi()`.
- `ui/` - Vue компоненты, могут представить фичу и как то отреагировать на его действие
  (например кнопки удаления, формы, диалоговые окна)
- `index.ts` - Реэкспорт публичного API
