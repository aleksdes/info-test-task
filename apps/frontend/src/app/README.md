# Ответственность
В зоне ответственности слоя находится все, что нужно приложению для инициализации.
Этот слой может импортировать из слоя `shared/`, и слоя композиции `widgets/` остальные слои функциональные 
и не могут участвовать в конфигурации приложения, поэтому импорт из слоев `entities/, features/` запрещен 

# App.vue
Главный компонент, содержит разметку роутера и разметку макета приложения
