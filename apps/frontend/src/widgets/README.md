Слой в свободной форме. Структура файлов в слое не имеет строгих требований, она может быть любой.

| Cross импорт между срезами              | Импорт других слоев                                            | Структура                                      |
|-----------------------------------------|----------------------------------------------------------------|------------------------------------------------|
| <span style="color:red">Запрещен</span> | <span style="color:yellow">Только `features` и `shared`</span> | <span style="color:green">Произвольная</span>  |

# Файловая структура среза
В приложении принята такая структура

- `Widget1.vue` - Компонент виджета который объединяет фича компоненты
- `index.ts` - Реэкспорт компонентов

Может содержать директории с вложенными файлами декомпозиции виджета

# Ответственность
Вспомогательный слой, тут можно объединить логически связанные сущности воедино в один виджет,
например фичи корзины можно объединить в данном слое, получив простой компонент виджета в управлении
