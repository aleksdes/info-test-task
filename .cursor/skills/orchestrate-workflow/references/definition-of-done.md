## Definition of Done (DoD) для этого репозитория

### Минимум (почти всегда)

- ✅ `npm run lint`
- ✅ `npx vue-tsc --noEmit`
- ✅ `npm run test:unit`

### Когда обязательно добавлять build

Если менялись:

- Vite/Vue конфиги, плагины, алиасы
- глобальные стили/тема
- роутинг, lazy‑imports, code splitting
- публичные API слайсов (массовые экспорты/импорты)

то также:

- ✅ `npm run build`

### Когда обязательно добавлять integration-тесты

Если менялись:

- UI/поведение компонентов (рендер, интеракции, состояния)
- контракты между `ui`/`model`/`api` внутри слайса
- код, завязанный на `msw`/моки транспорта в интеграционном окружении

то также:

- ✅ `npm run test:integration`
